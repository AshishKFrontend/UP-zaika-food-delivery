Welcome to **UP Zaika**, a premium, fully responsive food delivery web application built entirely with Vanilla web technologies. It features a modern Glassmorphism UI, smooth animations, and a backend-less checkout system!

🌍 **Live Demo:** [Click here to view live](https://ashishkfrontend.github.io/UP-zaika-food-delivery/)

---

## ✨ Key Features

* 🎨 **Modern Glassmorphism UI:** Stunning frosted glass effects with floating animations.
* 🌙 **Dark/Light Mode Toggle:** Seamless theme switching with local storage persistence.
* 🛒 **Dynamic Shopping Cart:** Real-time calculation of subtotal, 5% GST, and applying discount promo codes.
* 🔍 **Live Search & Filtering:** Instantly search for food items or filter them by category (Veg, Non-Veg, Fast Food, etc.).
* 📧 **Backend-less Checkout:** Order details, user login, and newsletter subscriptions are sent directly to the owner's email using the **FormSubmit API** (No server needed!).
* 🧑‍💻 **User Authentication Avatar:** Users can log in, and their initials appear as a custom avatar using `localStorage`.
* 📱 **Fully Responsive:** Includes a mobile-friendly hamburger menu and grid layout.
* 😻 **Lottie Animations:** High-quality, lightweight cat animations for better user engagement.

---

## 🛠️ Tech Stack Used

* **Frontend:** HTML5, CSS3 (Custom Variables, Flexbox, CSS Grid)
* **Scripting:** Vanilla JavaScript (ES6+)
* **APIs & Libraries:** * [FormSubmit](https://formsubmit.co/) (For sending emails without a backend)
  * [LottieFiles](https://lottiefiles.com/) (For JSON animations)
  * [Remix Icons](https://remixicon.com/) (For sleek social icons)
